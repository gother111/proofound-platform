# Source Of Truth Alignment

## Aligned

- Proof-first: Start from CV remains optional first-proof scaffolding, not profile activation or ATS intake.
- Privacy-first: session ownership, consent, redaction, no raw prompt logging, and draft-only review are preserved.
- No scoring/ranking: schemas, tests, UI copy, and provider policy avoid candidate evaluation language.
- Progressive reveal: nothing is shown to organizations from this flow.
- Canonical proof: CV output does not create verified proof or trust state.
- Skill suggestions: canonical mappings are review-only and carry no proficiency, trust lift, matching lift, or verification state.
