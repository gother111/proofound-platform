# Project Change Entry

- Date/time (UTC): 2026-03-06T09:09:36Z
- Branch: master
- Base commit: d3a4303d

What changed:

- Added automatic Atlas verification for unresolved CV skills in `src/lib/expertise/atlas-skill-verifier.ts` and wired it into the wizard suggest route so unresolved candidates arrive with verified top Atlas guesses.
- Unified wizard and JD manual-search behavior around the same Atlas verification URL builder and scoring path through `src/components/expertise/cv-import/taxonomy-search.ts`, `CvImportWizard.tsx`, `CVJDAutoSuggest.tsx`, and `manual-suggestion.ts`.
- Simplified unresolved review cards in `SkillReviewPanel.tsx` to show top auto-verified guesses by default and keep manual search as a refinement path.
- Expanded regression coverage for taxonomy verification, wizard/JD auto-refresh, and route orchestration in the updated `tests/api/*`, `tests/ui/*`, and `tests/lib/*` files.

Why:

- Users were forced to manually search Atlas for skills that were already discoverable by the taxonomy endpoint.
- Matching quality needed a shared verifier so automatic suggestions and manual replacement used the same ranking and ambiguity rules.
- The review UX needed to reduce work for unresolved skills without silently auto-approving weak matches.
