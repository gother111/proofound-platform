# Archived Evidence-Pack Export

This folder preserves the old donor/investor evidence-pack export experiments.

The locked MVP corridor uses Proof Packs, public portfolio exports, organization trust pages,
assignment/review workflows, reveal consent, interview decisions, and engagement verification.
Donor/investor evidence-pack PDF export is not an active launch surface, and no active MVP route or
test should imply that an evidence-pack API endpoint exists.
