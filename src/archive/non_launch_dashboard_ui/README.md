# Archived Customizable Dashboard UI

The old customizable/draggable individual and organization dashboard shells, plus their unmounted widget-card layer, live here as historical implementation context.

This shell is outside the locked MVP corridor. Active launch surfaces use focused proof-first home, organization home, assignment, review, verification, reveal, interview, decision, privacy, and launch-ops surfaces instead of a generic customizable dashboard or loose card grid.

Do not move these modules back into active `src/components/dashboard` or `src/app/app/*/home` unless the locked MVP source of truth changes. The active dashboard utility surface is limited to shared skeleton/chip helpers that mounted pages still import.
