# Non-launch landing variants

This archive preserves retired landing-section variants that are not imported by the active Proofound homepage.

The active public landing route is `src/components/ProofoundLanding.tsx` plus the scrollytelling, final CTA, and footer sections it imports. Retired variants must stay out of active `src/components/landing/sections/` unless they are reworked against the locked MVP corridor and verified with Browser.

- `hero-variants/HeroArtifact.archived.tsx` is a retired hero artifact variant that still shows legacy `/p` sample-link styling. It is preserved only as non-launch visual history.
