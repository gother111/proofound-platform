# Archived Admin Analytics UI

The old broad admin analytics, metrics, fairness, performance, user, and organization dashboards live here as historical implementation context.

These modules are outside the locked MVP corridor. Active admin launch ops are limited to the protected admin shell, verification queue, audit log, launch/status surfaces, and narrow internal diagnostics that are explicitly classified in route-surface policy.

Do not move these modules back into active `src/components` or `src/lib` unless the locked MVP source of truth changes.
