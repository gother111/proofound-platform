# Archived LinkedIn Scraper

Automated LinkedIn scraping and external LinkedIn verification checks are outside
the locked launch MVP corridor. Active launch code may preserve read-only
historical LinkedIn fields for compatibility, but it must not launch Playwright,
scrape LinkedIn profiles, call LinkedIn verification APIs, or create public trust,
matching, reveal, or intro lift from LinkedIn state.
