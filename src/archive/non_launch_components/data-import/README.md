# Archived Data Import Components

Settings data import is outside the locked launch MVP corridor. The active privacy
surface keeps owner export, data inventory, audit history, visibility controls, and
delete-account paths, but it does not expose import/restore workflows that call the
archived `/api/user/import` endpoint.

These components are preserved only as historical context for the retired import flow.
